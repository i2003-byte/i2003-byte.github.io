# Go Live Skill

When this skill is invoked:

1. Check that all changes are committed and pushed
2. Show the PR creation URL for the user to merge
3. Provide clear instructions on how to merge
4. Give the final live GitHub Pages URL where the page will be available

This skill helps you quickly get your changes live on GitHub Pages.
