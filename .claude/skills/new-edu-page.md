# New Educational Page Skill

When this skill is invoked:

1. Ask the user what concept they want to explain (physics, math, science, etc.)
2. Create a mobile-friendly HTML page with:
   - p5.js for interactive visualizations
   - Kid-friendly design with colorful gradients
   - Responsive layout that works on all devices
   - Touch support for mobile devices
   - Educational content with examples
3. Commit the new page to a new feature branch
4. Push the branch and provide instructions for merging

This skill helps you quickly create new educational content pages.
